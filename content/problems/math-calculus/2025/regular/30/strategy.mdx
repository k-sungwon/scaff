## 풀이 시작 전 30초 - 머릿속으로 정리할 것

### 1. 역함수 F가 절댓값으로 주어짐

→ 당황 금지. $F$가 증가함수니까 부호만 붙이면 됨.

### 2. 직선과 곡선의 교점 개수 $g(m)$

→ 구간별로 나눠서 세야겠다:
- **왼쪽** $x < -2$: $f(x) = -1 - \ln(-x-1)$ (로그)
- **중앙** $-2 \le x \le 2$: 명시적 식 없지만 연속·증가
- **오른쪽** $x > 2$: $f(x) = 1 + \ln(x-1)$ (로그)

### 3. $g(m)$이 불연속인 점

→ 교점이 합쳐지는 순간 = 접선 조건 $f'(x_0) = m$ 찾기

### 4. 최종 식에 $\left(\frac{\ln b}{b}\right)^2$ 있음

→ $b$를 정확한 숫자로 구할 필요 없이 관계식으로 처리 가능할 듯

```
   ↓
3. 구간별로 교점 개수 세기
   - 오른쪽: 0 < m < 1일 때 1개
   - 왼쪽: 접선 조건으로 m = b 찾기
   - 중앙: 사잇값정리
   ↓
4. 전체 g(m) 합치기
   ↓
5. 불연속점 a, b 확정
   ↓
6. 관계식으로 최종 계산
```
